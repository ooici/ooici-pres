Getting Started

Welcome to Release 1 of the Integrated Observatory Network (ION). If this is your first visit, you have the option to start using ION as a guest or to create a free ION account. Using ION as a guest allows you to browse all of ION's features, while creating an account allows you to set up notifications, register your own data resources, and maintain any settings for your next visit.

If you have questions, check the help options or contact us at contactooici@oceanobservatories.org

 
ION Release One

Developed by the Cyberinfrastructure Implementing Organization, Release 1 (R1) provides an operational data distribution network to numerical modelers and data analysts in real-time through a common scalable infrastructure. R1 incorporates a publish/subscribe messaging architecture, and incorporates the the behind the scenes infrastructure that will ensure scalability, security and fault tolerance for ION as it matures over the course of the development of the OOI. R1is the first increment of the core infrastructure that will connect and coordinate the operations of the three OOI marine components with the scientific and educational pursuits of the oceanographic research communities.

With R1 of ION, you can browse, search for and view metadata for registered data, request notifications about changes to data resources, register your own data resources and create and manage your ION account.
